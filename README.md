# WebKyu

这个是一个通过URLProtocol实现的轻量的浏览器，用于读取缓存的图片，当然如果愿意的话，也可以读取html，js等等，各种各样的文件，目前还没有支持读取视频。

# 开源
因为最近苹果的政策调整，所以这个产品就无法上架了，喜欢的朋友可以当做技术交流之用。
